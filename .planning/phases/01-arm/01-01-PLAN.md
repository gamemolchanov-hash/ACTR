<objective>
Переключить data-layer витрины ACTR с OMS API на ARM API так, чтобы каталог (листинг,
категории, карточка товара, картинки) рендерился из ARM, при этом дизайн и компоненты AC
не меняются. `X-Storefront-Key` держится server-side.

Purpose: первый и фундаментальный шаг — без рабочего ARM-каталога остальные фазы (корзина,
чекаут, ЛК) некуда вешать.
Output: ARM-прокси (route-handler), адаптер ARM→Product, перепроводка fetch-функций.
</objective>
